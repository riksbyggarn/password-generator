# Password Generator 🔑

## Demo

_Add a link here if you publish the project online!_

## Features

- Generate secure, random passwords directly in the browser
- Choose password length with a slider
- Two different passwords are generated at the same time
- Click a password to copy it to the clipboard ("Copied!" is shown)
- Simple and responsive design

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES Modules)

## Installation

1. Clone the repo:
   `git clone <your-repo-url>`
2. Open `index.html` in your web browser.

## Usage

- Select the desired password length with the slider.
- Click "Generate passwords" to create new passwords.
- Click a password to copy it.
